📡 IP Pinger

A Windows command-line Python tool used to test network connectivity and measure response times by sending ICMP ping requests to a target IP address.

This project is designed to help learn basic networking concepts and how latency and packet response work.

---

Features

- Ping IPv4 addresses
- Multiple ping requests
- Response time tracking
- Average latency calculation
- Simple command-line interface

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter the target IP address when prompted. The tool will send ping requests and display the response results.

---

Future Improvements

- Add domain name support (DNS resolution)
- Export ping results to a file
- Graph latency over time
- Improve error handling for unreachable hosts
- Add continuous ping mode
