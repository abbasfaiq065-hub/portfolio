🌍 Subdomain Enumerator

A Windows command-line Python tool that discovers publicly accessible subdomains for a target domain.

This project was built to learn domain enumeration techniques, DNS concepts, and information gathering using Python. It is intended for educational purposes and authorized security assessments.

---

Features

- Discover publicly accessible subdomains
- Fast enumeration using predefined methods
- Clean and easy-to-read command-line output
- Lightweight and beginner-friendly
- Supports authorized reconnaissance workflows

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter the target domain when prompted. The tool will enumerate publicly accessible subdomains and display the results.

---

Future Improvements

- Add support for additional data sources
- Export results to TXT or JSON
- Improve enumeration speed
- Detect duplicate results automatically
- Add optional DNS record lookups
