🔎 Port Scanner

A Windows command-line Python tool that scans a predefined list of TCP ports on a target host to check whether they are open or closed.

This project is designed to help learn basic network scanning concepts and how services operate on different ports.

---

Features

- Scans a predefined set of TCP ports (defined in the script)
- Detects open and closed ports
- Fast and simple execution
- Clean command-line output
- Beginner-friendly networking project

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter the target IP address or domain when prompted.
The scanner will check the ports defined in the script and display which ones are open.

---

Future Improvements

- Add custom port input from user
- Implement multithreading for faster scanning
- Add service detection (banner grabbing)
- Export scan results to a file
