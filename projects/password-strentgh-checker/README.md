✅ Password Strength Checker

A Windows command-line Python tool that evaluates the strength of a password using common cybersecurity best practices.

This project was built to learn password security concepts and help users create stronger, more secure passwords.

---

Features

- Analyze password strength
- Check password length
- Detect uppercase and lowercase letters
- Detect numbers
- Detect special characters
- Provide a strength rating
- Display recommendations for improving password security
- Simple and easy-to-use command-line interface

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter a password when prompted. The tool will analyze the password and display its strength along with recommendations for improvement.
