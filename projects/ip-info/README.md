🌐 IP Info

A Windows command-line Python application that retrieves detailed information about IPv4 and IPv6 addresses. This project was created to help learn networking concepts and working with IP-related data using Python.

---

Features

- IPv4 and IPv6 lookup
- ISP information
- Country, region, and city
- ASN information
- Time zone information (if available)
- Public IP support
- Clean command-line interface

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter the IP address when prompted. The tool will retrieve and display the available information.

---

Future Improvements

- Export results to JSON
- Save results to a text file
- Display additional network information
- Improve error handling
- Add support for multiple IP lookups
