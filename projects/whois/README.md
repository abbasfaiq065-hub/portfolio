📖 WHOIS Lookup

A Windows command-line Python tool that retrieves publicly available WHOIS registration information for domain names.

This project was built to learn how domain registration records work and how WHOIS data can be used for information gathering during authorized security assessments.

---

Features

- Domain WHOIS lookup
- Registrar information
- Domain registration date
- Domain expiration date
- Last updated date
- Name server information
- Domain status information
- Clean command-line interface

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter a domain name when prompted. The tool will retrieve and display the available WHOIS registration information.
