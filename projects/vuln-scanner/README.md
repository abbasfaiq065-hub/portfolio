🛡️ Vulnerability Scanner

A Windows command-line Python tool that performs basic security assessments against websites by testing for common web vulnerabilities and security misconfigurations.

This project was built to learn web application security, HTTP requests, and automated vulnerability testing using Python.

---

Features

- Tests for common web vulnerabilities
- XSS testing
- SQL injection testing
- Security header analysis
- HTTP response inspection
- Simple command-line interface
- Easy-to-read scan results

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script and enter the target website URL when prompted. The scanner will perform its predefined security checks and display the results.
