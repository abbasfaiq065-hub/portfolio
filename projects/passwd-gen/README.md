🔐 Password Generator

A Windows command-line Python tool that generates strong, random passwords based on user-defined preferences.

This project was built to learn secure password generation techniques and improve Python programming skills while promoting good password security practices.

---

Features

- Generate secure random passwords
- Custom password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Fast and easy-to-use command-line interface

---

Requirements

- Windows 10 or Windows 11
- Python 3.10 or newer

---

Usage

Run the Python script, choose the desired password length and character options, then the tool will generate a strong random password.

---

Future Improvements

- Generate multiple passwords at once
- Copy generated password to the clipboard
- Save generated passwords to a file
- Add password strength estimation
- Create a graphical user interface (GUI)
